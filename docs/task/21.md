tests/e2e/payment-flow.spec.ts

Because full live generation may depend on LLM and Supabase state, use one of two approaches.

Use a test seed report in Supabase test DB.

Seed a report with:

result_token = e2e-ready-unpaid
generation_status = READY

preview_text = test preview text
