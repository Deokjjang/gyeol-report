# Task 15 — Create Report API
## Context
We are building **결리포트 / Gyeol Report**, a mobile-first Saju × MBTI self-narrative report service.
Tasks 02~14 created the core deterministic pipeline pieces:
- Saju calculation
- MBTI profiles
- Saju tag extraction
- Saju-MBTI comparison
- Interpretation block selection
- Report prompt/LLM generation wrapper
- Report validation filter
This task creates the first server API route for report creation.
The API must:
```txt
사용자 입력
→ validation
→ report row 생성
→ 사주 계산
→ MBTI profile
→ 사주 태그 추출
→ 사주-MBTI 비교
→ 해석 블록 선택
→ LLM 리포트 생성
→ DB 저장
→ preview URL 반환

Do not implement frontend pages, payment, Toss integration, checkout, or full report unlocking in this task.

2. Result Token Utility

Create:

Export:

export function createResultToken(): string;

Requirements:

* Must be URL-safe.
* Must have at least 128-bit entropy.
* Use Node crypto.
* Do not use Math.random.
* Token should be suitable for /report/preview/[token].

Recommended implementation:

import crypto from "crypto";
export function createResultToken(): string {
  return crypto.randomBytes(24).toString("base64url");
}

24 bytes = 192-bit entropy.

4. Create Report Pipeline

Create:

src/lib/reports/createReport.ts

Export:

import type { LlmClient } from "@/lib/llm/client";
import type {
  CreateReportRequest,
  CreateReportPipelineResult,
} from "./types";
export type CreateReportDependencies = {
  db: ReportDbPort;
  llmClient: LlmClient;
  appUrl: string;
};
export type ReportDbPort = {
  insertInitialReport(input: InsertInitialReportInput): Promise<{ reportId: string }>;
  updateReportSuccess(input: UpdateReportSuccessInput): Promise<void>;
  updateReportFailure(input: UpdateReportFailureInput): Promise<void>;
  insertReportEvent(input: InsertReportEventInput): Promise<void>;
};
export type InsertInitialReportInput = {
  resultToken: string;
  request: NormalizedCreateReportRequest;
};
export type UpdateReportSuccessInput = {
  reportId: string;
  sajuResult: unknown;
  analysisTags: unknown;
  selectedBlocks: unknown;
  previewText: string;
  fullReport: unknown;
};
export type UpdateReportFailureInput = {
  reportId: string;
  errorMessage: string;
};
export type InsertReportEventInput = {
  reportId?: string;
  eventType: string;
  payload?: Record<string, unknown>;
};
export type NormalizedCreateReportRequest = {
  birthDate: string;
  birthTime?: string;
  birthTimeUnknown: boolean;
  calendarType: "SOLAR" | "LUNAR";
  isLeapMonth?: boolean;
  gender: "MALE" | "FEMALE" | "OTHER_OR_UNSPECIFIED";
  mbti: MbtiType;
};
export async function createReportPipeline(
  request: CreateReportRequest,
  deps: CreateReportDependencies,
): Promise<CreateReportPipelineResult>;

6. Input Normalization

Inside createReportPipeline:

1. Validate and normalize request.
2. Normalize MBTI with normalizeMbtiType().
3. Use only normalized request downstream.

Validation rules:

birthDate

Required.

Format:

YYYY-MM-DD

If invalid:

Invalid birth date format. Expected YYYY-MM-DD.

birthTime

If birthTimeUnknown === false, birthTime is required.

If invalid:

Invalid birth time format. Expected HH:mm.

calendarType

Allowed:

SOLAR
LUNAR

If invalid:

Invalid calendar type.

gender

Allowed:

MALE
FEMALE
OTHER_OR_UNSPECIFIED

If invalid:

Invalid gender.

mbti

Normalize invalid values to:

UNKNOWN

Do not throw for invalid MBTI.

7. Pipeline Steps

createReportPipeline() must run in this order:

1. normalize input
2. create resultToken
3. insert initial report row
4. insert REPORT_CREATED event
5. calculateSaju()
6. insert SAJU_CALCULATED event
7. get MBTI profile and traits
8. extractSajuTags()
9. compareSajuMbti()
10. insert TAGS_EXTRACTED event
11. selectInterpretationBlocks()
12. insert BLOCKS_SELECTED event
13. build ReportGenerationInput
14. generateReport()
15. update report success
16. insert PREVIEW_GENERATED event
17. insert FULL_REPORT_GENERATED event
18. return CreateReportPipelineResult

If any step after initial insert fails:

update report failure
insert REPORT_ERROR event
rethrow error

If failure happens before initial insert:

throw error

No DB update is possible before reportId exists.
