# Task 11 — Saju-MBTI Comparison
## Context
We are building **결리포트 / Gyeol Report**, a mobile-first Saju × MBTI self-narrative report service.
Task 10 implemented deterministic Saju structure tag extraction.
This task implements deterministic comparison between:
```txt
Saju structure tags
MBTI trait tags

V1 principle:

Saju = structural tendency data
MBTI = user self-perception data
Overlap = strong repeated pattern
Conflict = difference between structural tendency and self-perception

Do not implement interpretation block selection, report generation, API routes, UI, payment, or LLM logic in this task.

Relevant SSOT:

* docs/ssot/01_PRODUCT_SSOT.md
* docs/ssot/03_REPORT_SPEC.md
* docs/ssot/05_DEVELOPMENT_PLAN.md
