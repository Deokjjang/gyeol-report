5. getPreviewReport() Behavior

Token Validation

If token is empty or whitespace, throw:

Invalid result token.

DB Not Found

If no report found, throw:

Report not found.

Response

Return:

{
  resultToken,
  paymentStatus,
  generationStatus,
  previewText,
  lockedSections: LOCKED_REPORT_SECTIONS,
  price: getCurrentReportPrice(now)
}

Important

Do not return:

full_report
saju_result
analysis_tags
selected_blocks
birth_date
birth_time
gender
mbti

Preview API must not leak private/internal data.
