# Task 05 — Hour Pillar Calculation
## Context
We are building **결리포트 / Gyeol Report**, a mobile-first Saju × MBTI self-narrative report service.
Task 04 implemented deterministic year pillar and month pillar calculation.
This task implements **hour pillar calculation only**.
Do not implement lunar conversion, five elements, ten gods, hidden stems, relations, report generation, API routes, UI, payment, or LLM logic in this task.
Relevant SSOT:
- `docs/ssot/02_SAJU_CALC_SPEC.md`
- `docs/ssot/05_DEVELOPMENT_PLAN.md`
## Goal
Implement deterministic hour pillar calculation from:
```txt
birthTime
dayStem
birthTimeUnknown

V1 policy:

시지 = 2-hour branch range
子 hour = 23:00~00:59
시천간 = 일간 기준 子시 천간표 + 시간지 순환
출생시간 모름 = 시주 없음
자시 일자 변경 = V1 기본 조자시, but this task only calculates hour pillar from provided dayStem/time

2. Birth Time Parsing

Accept only:

HH:mm

Required behavior:

* "00:00" valid
* "23:59" valid
* "24:00" invalid
* "7:00" invalid
* "07:0" invalid
* "07:00" valid
* "12:30" valid

Error message:

Invalid birth time format. Expected HH:mm.

Use strict validation.

Do not rely on loose Date parsing.

⸻

3. Hour Branch Ranges

Use existing constant from constants.ts if present:

HOUR_BRANCH_RANGES

Policy:

Time Range	Branch
23:00 ~ 00:59	子
01:00 ~ 02:59	丑
03:00 ~ 04:59	寅
05:00 ~ 06:59	卯
07:00 ~ 08:59	辰
09:00 ~ 10:59	巳
11:00 ~ 12:59	午
13:00 ~ 14:59	未
15:00 ~ 16:59	申
17:00 ~ 18:59	酉
19:00 ~ 20:59	戌
21:00 ~ 22:59	亥

Important:

子 hour crosses midnight and covers both 23 and 0.

⸻

4. Hour Stem Calculation

Use existing constant from constants.ts:

HOUR_STEM_START_BY_DAY_STEM

This maps day stem to 子 hour stem.

Formula:

hourBranchIndex = index of hour branch in EARTHLY_BRANCHES
startStem = HOUR_STEM_START_BY_DAY_STEM[dayStem]
startStemIndex = STEM_INDEX[startStem]
hourStem = HEAVENLY_STEMS[(startStemIndex + hourBranchIndex) % 10]

Return:

{
  stem: hourStem,
  branch: hourBranch
}

⸻
