# Task 02 — Saju Types and Constants
## Context
We are building **결리포트 / Gyeol Report**, a mobile-first Saju × MBTI self-narrative report service.
Task 01 created the initial Supabase schema and DB client placeholders.
This task starts the deterministic Saju calculation engine by creating only the shared types and fixed constants.
Do not implement calculation functions in this task.
Relevant SSOT:
- `docs/ssot/02_SAJU_CALC_SPEC.md`
- `docs/ssot/04_TECH_SPEC.md`
- `docs/ssot/05_DEVELOPMENT_PLAN.md`

Create the foundational Saju type definitions and constant tables used by later calculation modules.
