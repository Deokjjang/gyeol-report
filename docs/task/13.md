3. Prompt Requirements

The prompt must explicitly include:

1. Product identity
2. LLM role limitation
3. Saju summary
4. MBTI summary
5. Analysis tags
6. Selected blocks by section
7. Required JSON output schema
8. Tone guide
9. Forbidden content rules
10. V1 exclusion rules

⸻

4. Product Identity Text

Prompt must include this concept:

결리포트는 사주와 MBTI를 결합해 사용자의 타고난 구조와 자기인식이 어디서 겹치고 어긋나는지 읽어주는 자기서사 리포트 서비스다.

⸻

5. LLM Role Limitation

Prompt must include exact or equivalent instruction:

너는 사주를 계산하지 않는다.
너는 MBTI를 진단하지 않는다.
너는 제공된 계산 결과, 태그, 해석 블록만 사용해 문장을 편집한다.
입력에 없는 명리학 요소, 신살, 용신, 대운, 세운, 궁합을 새로 만들지 않는다.

⸻

6. Tone Guide

Prompt must include:

부드럽게 말하되, 회피하지 않는다.
위로하되, 무조건 좋은 말만 하지 않는다.
약점을 비난하지 않고 패턴으로 설명한다.
명리학 용어는 근거로 쓰되, 최종 문장은 현대어로 번역한다.

13. Create src/lib/llm/client.ts

Create a server-only LLM wrapper.

Export:

import "server-only";
export type LlmGenerateJsonInput = {
  prompt: string;
  temperature?: number;
  maxTokens?: number;
};
export type LlmClient = {
  generateJson(input: LlmGenerateJsonInput): Promise<unknown>;
};
export function createLlmClient(): LlmClient;

Implementation

For this task, implement one of two acceptable options.

Option A — Real OpenAI-compatible client

Use LLM_API_KEY and LLM_MODEL.

Do not hardcode provider-specific behavior deeply.

Requirements:

* Throw clear error if env vars missing.
* Server-only.
* Return parsed JSON.
* Timeout handling preferred.

Option B — Stub client with explicit not-implemented error

Acceptable for this task:

export function createLlmClient(): LlmClient {
  return {
    async generateJson() {
      throw new Error("LLM client is not configured yet.");
    },
  };
}

Recommended for Task 13: Option B unless the project already has an LLM provider dependency.

Reason:

This task should lock prompt and interface first.
Real provider integration can be swapped without touching report pipeline.

14. Create generateReport.ts

Create:

src/lib/analysis/generateReport.ts

Export:

import type { GeneratedReport, ReportGenerationInput } from "./reportTypes";
import type { LlmClient } from "@/lib/llm/client";
export type GenerateReportOptions = {
  llmClient?: LlmClient;
};
export async function generateReport(
  input: ReportGenerationInput,
  options?: GenerateReportOptions,
): Promise<GeneratedReport>;

Behavior

1. Build prompt with buildReportPrompt(input).
2. Use provided options.llmClient if present.
3. Otherwise use createLlmClient().
4. Call llmClient.generateJson({ prompt, temperature: 0.4, maxTokens: ... }).
5. Parse/validate the returned shape minimally.
6. Return GeneratedReport.
