For now, disable LUNAR option or show notice.

Best for this task:

음력 입력은 준비 중입니다. 현재는 양력만 지원합니다.

Implementation options:

Preferred

Render LUNAR option disabled.

음력 — 준비 중

Do not allow users to submit LUNAR unless API can handle it.

Task 15 currently throws for LUNAR.
