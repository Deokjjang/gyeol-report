5. Ten God Grouping

Group Ten Gods into 5 high-level groups.

const TEN_GOD_GROUP: Record<TenGod, TenGodGroup> = {
  比肩: "PEER",
  劫財: "PEER",
  食神: "EXPRESSION",
  傷官: "EXPRESSION",
  偏財: "WEALTH",
  正財: "WEALTH",
  偏官: "OFFICER",
  正官: "OFFICER",
  偏印: "RESOURCE",
  正印: "RESOURCE",
};

Compute tenGodGroupScores from:

result.tenGods.distribution

Score Rules

Sum distribution values by group.

Example:

偏財 1.2 + 正財 0.6 = WEALTH 1.8

⸻

6. Ten God Group Labels

Use deterministic thresholds:

MISSING = 0
WEAK = > 0 and < 0.8
STRONG = >= 2.0

For each group:

PEER
EXPRESSION
WEALTH
OFFICER
RESOURCE

Generate tags:

PEER_MISSING / PEER_WEAK / PEER_STRONG
EXPRESSION_MISSING / EXPRESSION_WEAK / EXPRESSION_STRONG
WEALTH_MISSING / WEALTH_WEAK / WEALTH_STRONG
OFFICER_MISSING / OFFICER_WEAK / OFFICER_STRONG
RESOURCE_MISSING / RESOURCE_WEAK / RESOURCE_STRONG

No tag for medium range.

Thresholds are V1 heuristic and must remain deterministic.

⸻

7. Yin/Yang Tags

Map:

result.yinYang.label === "YIN_HEAVY" -> YIN_HEAVY
result.yinYang.label === "YANG_HEAVY" -> YANG_HEAVY
result.yinYang.label === "BALANCED" -> YIN_YANG_BALANCED

⸻

8. Relation Tags

From:

result.relations

Add:

STEM_COMBINATION_PRESENT if stemCombinations.length > 0
BRANCH_COMBINATION_PRESENT if branchCombinations.length > 0
BRANCH_CLASH_PRESENT if branchClashes.length > 0

⸻

9. Derived Narrative Signal Rules

Add derived tags based on combinations.

HIGH_PRESSURE

Add if:

OFFICER_STRONG exists

or

BRANCH_CLASH_PRESENT exists AND OFFICER is not missing

HIGH_STANDARD

Add if any:

OFFICER_STRONG
RESOURCE_STRONG
METAL_STRONG

LOW_RECOVERY

Add if any:

RESOURCE_MISSING
WATER_MISSING

EXPRESSION_CONSTRAINED

Add if:

EXPRESSION_MISSING

or:

FIRE_MISSING

GOAL_ORIENTED_STRUCTURE

Add if any:

WEALTH_STRONG
OFFICER_STRONG
EARTH_STRONG

STABILITY_ORIENTED_STRUCTURE

Add if any:

EARTH_STRONG
YIN_YANG_BALANCED + RESOURCE_STRONG

RELATIONSHIP_SENSITIVE_STRUCTURE

Add if any:

RESOURCE_STRONG
BRANCH_COMBINATION_PRESENT
YIN_HEAVY

INDEPENDENCE_ORIENTED_STRUCTURE

Add if any:

PEER_STRONG
DAY_MASTER_GAP_WOOD
DAY_MASTER_GYEONG_METAL
YANG_HEAVY

CHANGE_TENSION_STRUCTURE

Add if:

BRANCH_CLASH_PRESENT

⸻

10. Duplicate Handling

structureTags must not contain duplicates.

Return tags in stable order:

1. Day master tag
2. Element tags in result order
3. Ten God group tags in PEER, EXPRESSION, WEALTH, OFFICER, RESOURCE order
4. Yin/Yang tag
5. Relation tags
6. Derived narrative tags
