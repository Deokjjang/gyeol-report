# 결리포트 Product SSOT v0.1

Status: DRAFT  
Product Name: 결리포트  
Tagline: 사주와 MBTI로 읽는 나의 결  
Product Type: Mobile-first web report service  
Initial Price: 990 KRW launch price / 1,290 KRW regular price

---

## 1. Product Definition

결리포트는 사용자의 생년월일시 기반 만세력 사주 구조와 사용자가 직접 입력한 MBTI 성향을 결합하여, 사용자의 기질·성격 스타일·관계 패턴·일/직업 성향·금전 감각·보완 방향을 해석하는 초저가 자기서사 리포트 서비스다.

결리포트는 운명을 단정하는 서비스가 아니다.

결리포트는 다음을 제공한다.

- 사주 구조 기반 기질 해석
- MBTI 기반 자기인식 해석
- 사주와 MBTI의 공통점 분석
- 사주와 MBTI의 충돌점 분석
- 연애, 일, 직업, 금전, 인간관계에서 나타나는 성향 요약
- 사용자가 자기 패턴을 이해하고 보완할 수 있는 방향 제시

---

## 2. Core Positioning

결리포트의 핵심 포지션은 다음과 같다.

> 사주는 타고난 구조를 보고, MBTI는 사용자의 자기인식을 본다.  
> 두 해석이 겹치는 지점은 반복적으로 드러나는 강한 패턴이고, 어긋나는 지점은 사용자가 스스로 인식하는 모습과 실제 작동 방식의 차이로 해석한다.

결리포트는 정통 사주 상담, 심리검사, 의학적 진단, 투자 조언, 법률 조언이 아니다.

결리포트는 자기이해용 콘텐츠 상품이다.

## 3. Target Users

Primary Target:

- 10대 후반 ~ 30대
- 사주, MBTI, 성격 분석, 자기이해 콘텐츠에 관심 있는 사용자
- 연애, 일, 직업, 금전, 인간관계에서 자신의 성향을 알고 싶은 사용자
- 가볍게 결제 가능한 저가 개인화 리포트를 선호하는 사용자

Secondary Target:

- 친구, 연인, 지인과 결과를 비교하고 싶은 사용자
- 향후 궁합 리포트, 신년사주, 대운/세운 리포트로 확장 가능한 사용자

## 9. Birth Time Unknown Policy

출생시간을 모르는 사용자는 서비스를 사용할 수 있어야 한다.

### Default Behavior

- 출생시간 모름 선택 시 년주, 월주, 일주 중심의 삼주 리포트를 생성한다.
- 시주 기반 해석은 확정적으로 말하지 않는다.

### Optional Simulation

출생시간을 모르는 경우, MBTI 성향과 사주 원국의 부족/과다 구조를 비교하여 “시주 후보 시뮬레이션”을 제공할 수 있다.

단, 이 기능은 실제 출생시간을 대체하지 않는다.

Allowed wording:

> 출생시간이 확인되지 않았기 때문에 시주는 확정할 수 없습니다.  
> 다만 MBTI 성향과 사주 구조를 비교했을 때, 이런 기운이 보완되는 시주 후보가 자기서사적으로 더 잘 맞아 보일 수 있습니다.

Forbidden wording:

> MBTI로 당신의 태어난 시간을 맞혔습니다.  
> 당신의 실제 시주는 이것입니다.

## 10. Saju and MBTI Combination Principle

### Saju

사주는 만세력 기반 구조 데이터로 취급한다.

Used as:

- 일간
- 월주
- 일주
- 시주
- 오행 분포
- 십성 구조
- 음양 분포
- 지장간
- 합충 등 주요 관계

### MBTI

MBTI는 사용자가 직접 입력한 자기인식 데이터로 취급한다.

Used as:

- 사용자가 스스로 인식하는 성향
- 행동 선호
- 대인관계 선호
- 판단/인식 방식
- 에너지 방향성

### Combination

사주와 MBTI를 1:1로 강제 매핑하지 않는다.

Allowed:

- “이 사주 구조는 ENTJ 성향과 닮은 부분이 있습니다.”
- “MBTI로는 외향적 리더형에 가깝지만, 사주 구조상 내면 긴장도는 높은 편으로 해석됩니다.”
- “두 체계가 모두 목표지향성과 기준 의식을 강조합니다.”

Forbidden:

- “이 사주는 ENTJ입니다.”
- “갑목은 ENTJ입니다.”
- “편관이 있으면 무조건 ENTJ입니다.”
- “MBTI가 사주보다 정확합니다.”
- “사주가 MBTI보다 무조건 정확합니다.”

## 11. Report Tone

결리포트의 문체는 다음을 따른다.

> 부드럽게 말하되, 회피하지 않는다.  
> 위로하되, 무조건 좋은 말만 하지 않는다.  
> 약점을 비난하지 않고 패턴으로 설명한다.  
> 명리학 용어를 근거로 사용하되, 최종 문장은 현대어로 번역한다.

### Good Tone

- 따뜻함
- 자기이해
- 현실적 조언
- 핵심을 찌르는 문장
- 과장 없는 해석

### Bad Tone

- 공포 조장
- 운명 단정
- 지나친 칭찬
- 저주/불운 암시
- 질병/사망/사고 예언
- 결혼/이별/투자 단정

## 17. Success Metrics

MVP 성공 판단은 다음 지표로 본다.

### Primary Metrics

- 방문자 → 무료 미리보기 생성 전환율
- 무료 미리보기 → 결제 전환율
- 결제 완료율
- 결제 후 결과 열람 완료율

### Target Benchmarks

- 방문자 → 무료 미리보기 생성: 20% 이상
- 무료 미리보기 → 결제: 5~10% 이상
- 결제 오류율: 1% 이하
- 환불/불만율: 3% 이하

### Qualitative Signals

- “내 얘기 같다” 반응
- “친구/연인 것도 해보고 싶다” 반응
- 궁합 리포트 요청
- 연애/직업/금전 심화 요청

## 18. Development Principle

V1 개발은 다음 원칙을 따른다.

- 웹앱 우선
- 앱 개발 금지
- 회원가입 강제 금지
- 기능 확장보다 리포트 품질 우선
- 만세력 계산 정확도 우선
- LLM은 해석자가 아니라 문장 편집자
- 사주 계산과 태그 판단은 코드/룰 기반
- 결제/리포트 플로우를 먼저 완성
- 대운/세운/궁합은 후속 상품으로 분리
